# **App Name**: VPK Assembler

## Core Features:

- File Discovery: Automatically discover and list all .vpk files on connected drives.
- Metadata Extraction: Extract metadata for each .vpk file, including name, date, time, drive location, and last used date.
- Data Storage: Store file system metadata within a Cloud SQL database, including name, date, time, drive location and last used date.
- Sortable Display: Display .vpk files in a sortable table format based on extracted metadata (name, date, time, drive, last used).
- File Preview: Offer an automatically-generated, rich preview, if filetype allows.
- Intelligent Search: AI tool to classify and label VPK files.
- Theme selection: Provide theme and color-coding based on file type

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to invoke trust and stability.
- Background color: Very light blue (#F0F8FF). to provide a subtle and unobtrusive backdrop.
- Accent color: Dark blue (#00008B), used selectively to highlight important elements and actions, ensuring they stand out against the light backdrop.
- Body and headline font: 'Inter', a sans-serif font, for a clean and modern interface. Great legibility.
- Use minimalist icons to represent file types and actions.
- A tabular layout will be employed to clearly display file information.
- Subtle transitions when sorting or filtering the file list.
